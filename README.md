## - Control: 
  
> **P1**: `W`, `S`, `A`, `D`
>
> **P2**: `↑`, `↓`, `←`, `→`
>
> **Pause**: `Space`
> 
> **Restart**: `Enter`
>
> **Full Screen**: `F11`
>
> **Exit**: `Escape`

## - Usage

1. Use python:
   
        > cd $(current_floder)
   
        > pip install requirements.txt
   
        > cd bebarball
   
        > python BeBarBall.py
    
2. Create executable:
    
    > see in _**build**_ folder
